# finance-tracker
A simple personal finance tracker written in python.
Includes a Flask API backend to use as future web or mobile app.

## features

-Add and delete transactions
-List all transactions
-View basic summary
-Edit transactions
-Storage using json

## Stack
-Python
-Dataclasses
-JSON
-Flask

## structure
-app.py
-main.py
-storage.py
-transactions.py
